# Saqlain Portfolio (Clone-style, with Research + Publications)

Next.js App Router + Tailwind + Framer Motion.

## Run
```bash
npm install
npm run dev
```

## Customize
- `data/content.ts` is the single source of truth for text/links.
- Optional: replace `public/resume.pdf` with your actual PDF.

## Animations
- Scroll reveal on every section
- Parallax glow blobs in the background
- Timeline cards slide in from alternating sides
- Skill bars animate when visible
